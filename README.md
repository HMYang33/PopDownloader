## 🔨 启动项目
0. 确保设备已经安装[Node.js](https://nodejs.org/)
1. 安装项目所需依赖
```bash
> npm install
```
2. 启动项目
```bash
> npm start
```
3. 访问网站，页面链接见控制台输出
```bash
> popDownloader@1.0.0 start
> node server.js

Server is running on http://localhost:3000     <———— 此 URL 即页面链接
```
